# Project 2: Network Packet Sniffer

Capture and analyse live network traffic — built from scratch in Python and demonstrated professionally with Wireshark.

## Versions

### Python (`python/`)
A command-line packet sniffer built with Scapy. Captures TCP, UDP, ICMP, and DNS traffic in real time with timestamps, protocol filtering, and log file output.

### Wireshark (`tools/`)
A professional walkthrough of live packet capture using Wireshark — interface selection, display filters (DNS, ICMP, HTTP), and saving captures for offline analysis.

## Skills Demonstrated

- Raw packet capture with Scapy
- Protocol layer parsing (Ethernet → IP → TCP/UDP/DNS)
- DNS query extraction
- Wireshark display filters
- Network interface analysis (`ip a`, `ip route`)
- Capture file management (`.pcapng`)

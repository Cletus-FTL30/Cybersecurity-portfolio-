# Cybersecurity Portfolio

A collection of Python-based cybersecurity tools built from scratch, alongside walkthroughs using industry-standard professional tools.

Each project has two versions:
- `python/` — built from scratch to demonstrate understanding of how the tool works
- `tools/` — using real professional tools used in the industry
